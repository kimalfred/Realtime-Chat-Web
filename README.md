## Free Chat API 
https://chatengine.io/

## Import

Backend

```
cd backend/
npm init
npm install express cors axios
npm install --save-dev nodemon
```

Frontend

```
npm create vite@latest
npm install
```

Frontend Chat Design
https://www.npmjs.com/package/react-chat-engine-advanced
```
npm i react-chat-engine-advanced
```

To run NodeJs

```
npm run start
```
To run ReactJs

```
npm run dev
```
